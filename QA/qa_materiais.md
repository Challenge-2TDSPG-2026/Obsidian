## 🗒️ Glossário

| Termo | Definição |
|---|---|
| Unit Test | Testa uma unidade isolada (método/classe) sem dependências externas |
| Integration Test | Testa a comunicação entre módulos/camadas/serviços |
| E2E Test | Simula o fluxo completo do usuário ponta a ponta |
| Mock | Substituto de dependência controlado em testes |
| Coverage | % do código coberto por testes automatizados |
| CI/CD | Pipeline de integração e entrega contínua |
| LGPD | Lei Geral de Proteção de Dados — impacta coleta e uso de dados pessoais |
| Compliance | Conformidade com leis, normas e políticas internas |
| DoD | Definition of Done — critérios para considerar uma tarefa concluída |
| Regression | Teste que verifica que mudanças não quebraram funcionalidades existentes |
